# AI Resume Scanner & Ranker

An AI-powered Resume Scanner and Candidate Ranking System built using Spring Boot, Java, HTML, CSS, and JavaScript.

## Features

- Upload multiple resumes (PDF)
- Extract resume content using Apache PDFBox
- Analyze candidate skills and experience
- Match resumes against job descriptions
- Generate AI-based feedback
- Rank candidates based on match score
- Modern responsive frontend UI

## Tech Stack

### Backend
- Java 17
- Spring Boot
- Maven
- Apache PDFBox
- OpenAI API

### Frontend
- HTML5
- CSS3
- JavaScript

## Project Structure

Backend/
Frontend/

## How to Run

### Backend
1. Open Backend project in Eclipse/STS.
2. Run `ResumeScannerApplication.java`.
3. Server starts on:
   http://localhost:8080

### Frontend
1. Open Frontend folder in VS Code.
2. Install Live Server extension.
3. Right-click `index.html`.
4. Click "Open with Live Server".
